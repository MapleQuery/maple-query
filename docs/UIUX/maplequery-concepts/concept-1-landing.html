The data is public. Using it isn't.

Access to government data has improved; usability hasn't kept pace. The friction lives in four places.

Fragmented

One topic is scattered across departments, datasets, and formats.

Inconsistent

Different structures, standards, gaps, and stale records.

Technical

Joining and cleaning needs tools and skills many newsrooms lack.

Time-consuming

Prep eats the hours meant for analysis and writing.

How it works

Ask, read the cited answer, trace every source

01

Ask in plain language

No SQL, no spreadsheets. Type a question the way you'd ask a colleague — MapleQuery finds, normalizes, and joins the right datasets.

02

Read a cited answer

Get a clear answer with tables and figures. Every number carries a footnote; when evidence is thin, MapleQuery says so instead of guessing.

03

Trace the evidence rail

A side panel shows the datasets behind the answer — coverage, freshness, completeness — with one-click links to the original records.
