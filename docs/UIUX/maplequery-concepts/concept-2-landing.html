How it works

Write a block, run a query, promote to report

01

Mix prose & queries

Type /query, /text, or /source to add blocks. Drag to reorder anytime.

02

Run for live results

Hit Run and a query block fills with an animated chart or table — plus an auto-generated citation underneath.

03

Promote to report

Assemble the blocks into a structured report and export — every figure carries its source.

Every figure stays sourced

  • Auto-citations

    Each result block writes its own source line you can click through.

  • Honest gaps

    Partial or redacted data is flagged, never quietly smoothed over.

  • Provenance kept on export

    Citations survive the trip to Google Docs or Word.
